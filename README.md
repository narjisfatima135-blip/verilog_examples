# Verilog RTL Design Examples

This repository contains a collection of professional Verilog RTL design examples used in digital design and VLSI workflows.

## 📌 Contents

- Combinational Circuits
- Sequential Circuits
- FSM Designs
- Arithmetic Units
- Testbenches

## 🛠 Tools Used
- Synopsys Design Compiler
- VCS (Simulation)
- Verdi (Debugging)

## 📂 Structure

| Module | Description |
|--------|------------|
| alu.v | Arithmetic Logic Unit (ALU) |
| dff.v | D Flip-Flop |
| counter.v | Synchronous Counter |
| fsm.v | Finite State Machine |
| tb_*.v | Testbenches |

## 🎯 Objective
To demonstrate RTL coding practices used in real ASIC/FPGA design flows.
